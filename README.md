# mods.de enhancer

Diese Chrome Erweiterung fügt einige Features zum mods.de Forum hinzu.

Features derzeit:

  * Verstecken der in-thread Werbung
  * Verstecken von Usern (Posts sind wieder einblendbar)

  * Anzeigen der Post-Nummer
